# Security Notes

`pharmacovigilance-signal-router` publishes synthetic biotech safety-signal and review-evidence data only.

It does **not** contain:

- live patient-identifiable safety data
- sponsor-confidential case narratives
- production safety-system credentials
- proprietary study or product data
- real regulatory submission packets

This repo is a public operator-surface proof for biotech and diagnostics workflows. It is not a clinical, regulatory, or safety certification claim.
